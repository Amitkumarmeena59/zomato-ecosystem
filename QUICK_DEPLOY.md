# ⚡ Bharatpur Bite - Quick Deploy (5 Minutes)

## 🚀 Super Fast Deployment Steps

### Step 1: Railway Backend (2 min)
```
1. Go: https://railway.app
2. Click: "Start a New Project"
3. Select: "Deploy from GitHub"
4. Choose: Amitkumarmeena59/zomato-ecosystem
5. Click: "Deploy Now"
6. Wait for deployment ✅
7. Copy the Public URL from Deployments tab
```

**Your Backend URL:** `https://your-railway-url.up.railway.app`

---

### Step 2: Vercel Frontend (2 min)
```
1. Go: https://vercel.com
2. Click: "New Project"
3. Import: Amitkumarmeena59/zomato-ecosystem
4. Add Environment Variable:
   Name: VITE_API_URL
   Value: https://your-railway-url.up.railway.app/api/trpc
5. Click: "Deploy"
6. Wait for deployment ✅
```

**Your Frontend URL:** `https://your-vercel-url.vercel.app`

---

### Step 3: Test (1 min)
```
1. Open: https://your-vercel-url.vercel.app
2. Click: "Login with Manus"
3. Go to: /admin-panel
4. Click: "Seed Dummy Restaurants"
5. Create a test order
6. Check all dashboards ✅
```
